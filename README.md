# myfolio
A full-stack, responsive portfolio built with React, JavaScript, and Tailwind CSS, powered by Node.js, Express, and MongoDB. Showcases projects, skills, and dynamic features like contact forms and project management with a focus on performance and scalability.

## MyFolio Starter

**Stack**
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express + MongoDB
- Docker: Compose for local development

**Quick start**
1. Install Docker Desktop.
2. Run `docker compose up --build`.
3. Frontend: `http://localhost:5173`
4. Backend health: `http://localhost:4000/health`

**Local (without Docker)**
1. Frontend
   - `cd frontend`
   - `npm install`
   - `npm run dev`
2. Backend
   - `cd backend`
   - `npm install`
   - `npm run dev`

**Environment**
- Backend env example in `backend/.env.example`.
